# PIXA Docs

This folder contains the Mintlify documentation site for PIXA.

The docs now focus on:

- installation for non-technical users and developers
- the PIXA overview and architecture
- the npm-published MCP package
- the PIXA Hub payment and routing model
- testing and contribution guidance

To preview locally:

```bash
mint dev
```

Keep the docs aligned with the current product shape:

- Algorand is the home wallet
- PIXA Hub handles routing and treasury-backed flows
- hosted browser pages handle approval-heavy actions
- AC2 is the direction for stronger session control
