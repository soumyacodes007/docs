# Spending Tools

PIXA is most useful when the user can see what happened after an agent spends money.

## `spending_report`

Use this tool to summarize spending activity.

Typical output should help answer:

- what was spent
- where it went
- which tool or service caused the spend
- how much budget remains

## Why this matters

Budget visibility is what turns a wallet into a control layer.

Without a spending report, the user only knows that money left the wallet.
With it, the user can understand the agent's behavior.

## Recommended usage

Call this tool after:

- a payment sequence
- a demo run
- a session that allowed multiple payments
- any flow where the user may ask, "What did the agent do?"
