# TypeScript Types

Complete reference for all TypeScript types used in the CeloRefer SDK.

## User Types

### UserInfo

```typescript
interface UserInfo {
  referralCode: string;
  referralCount: number;
  badgeTier: BadgeTier;
  registeredAt: number; // Unix timestamp
  referrer: Address | null;
}
```

### UserRewards

```typescript
interface UserRewards {
  totalEarned: bigint; // In wei
  available: bigint;   // In wei
  claimed: bigint;     // In wei
}
```

### ReferralTree

```typescript
interface ReferralTree {
  directReferrals: Address[];
  secondLevelReferrals: Address[];
}
```

## Badge Types

### BadgeTier

```typescript
type BadgeTier = 'Bronze' | 'Silver' | 'Gold' | 'Platinum';
```

### RewardRates

```typescript
interface RewardRates {
  level1: number; // Percentage for direct referrer
  level2: number; // Percentage for parent referrer
}
```

### BadgeRequirements

```typescript
interface BadgeRequirements {
  Bronze: { minReferrals: number; maxReferrals: number };
  Silver: { minReferrals: number; maxReferrals: number };
  Gold: { minReferrals: number; maxReferrals: number };
  Platinum: { minReferrals: number; maxReferrals: number };
}
```

### BadgeProgress

```typescript
interface BadgeProgress {
  currentTier: BadgeTier;
  nextTier: BadgeTier | null;
  referralsNeeded: number;
}
```

### PlatformStats

```typescript
interface PlatformStats {
  badgeDistribution: Record<BadgeTier, number>;
  totalUsers: number;
  totalReferrals: number;
}
```

## Quest Types

### Quest

```typescript
interface Quest {
  id: bigint;
  name: string;
  description: string;
  targetReferralCount: number;
  rewardAmount: bigint; // In wei
  startTime: number; // Unix timestamp
  endTime: number; // Unix timestamp
  isActive: boolean;
}
```

### QuestProgress

```typescript
interface QuestProgress {
```
