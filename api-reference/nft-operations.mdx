# DEX and Token Tools

These tools help PIXA move beyond simple transfers.

## `tinyman_swap`

Use this tool to swap assets through Tinyman.

Typical cases:

- convert between assets during a session
- rebalance a wallet before payment
- test liquidity-related flows on Algorand

## `create_token`

Use this tool to create a custom ASA token.

Typical cases:

- test a new asset flow
- represent a demo asset
- create a simple issued token for experimentation

## Guidance

Only use these tools when the user actually needs a swap or token operation.
They are not part of the basic payment path.
